# 工具的几个缺陷
1. 工具运行时不太好进行其它操作, 图像识别进度只能回到上一次的界面, (如果断了就进行重新运行)
2. 材料不能满就算是499都可以, 就是不能500, 识别不到被白色蒙版遮挡的材料 (直接丢弃一些物资)
3. 日活要打掉在开始挂机, 不然右边会出来邀请组队通知, 挡住图片识别 (要不做完日活, 要不就自己建个队伍就不会出现了)
4. 只能靠月卡自动采集, 无解决方案
5. 不是所有手套技能都能触发, 主要是我每抽到过, 没有图片资源也没法测试
6. 只测试过分辨率: 1080*1920, DPI 480 其它分辨率没测试过, 极大可能识图无法使用
7. 配方最好是满熟练度, 以及没有灵感触发的配方, 弹窗会导致识图失败. (自己关掉重新运行程序, 熟练度用完也就不触发了, 但是灵感触发的配方还是会弹窗)
